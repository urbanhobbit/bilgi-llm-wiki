## 🚀 Obsidian'ı Kullanmaya Başla

### 1. Obsidian'ı İndir
- Windows: https://obsidian.md/download/windows
- Mac: https://obsidian.md/download/macos
- Linux: https://obsidian.md/download/linux
- Mobile: iOS/Android App Store

### 2. Vault Aç
```bash
# Terminal'den
obsidian open /root/wiki

# Veya GUI'den
- Obsidian'u aç
- "Open folder as vault" seç
- /root/wiki klasörünü seç
```

### 3. Önerilen Plugins
1. **Dataview** - Dinamik içerik
2. **Templater** - Şablonlar
3. **QuickAdd** - Hızlı ekleme
4. **Calendar** - Log entegrasyonu
5. **Advanced Tables** - Tablolar

### 4. Graph View Kullan
- Sol panelden "Graph view" ikonuna tıkla
- Wikilink'lerin görsel bağlantılarını gör
- Tag renkleriyle kategorize et

### 5. Bülten İş Akışı
```
1. Yeni kaynak bul → raw/articles/klasörüne kaydet
2. Entity/concept sayfalarını güncelle
3. index.md ve log.md'yı güncelle
4. Bülten dosyasını ~/Bulten_Havuzu/cocuk_dijital/klasörüne kaydet
```
